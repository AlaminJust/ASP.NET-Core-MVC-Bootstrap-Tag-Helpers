jQWidgets Bootstrap Tag Helpers - Setup

In this tutorial, you will learn how to integrate the jQWidgets Bootstrap Tag Helpers into an existing project. We will be using Microsoft Visual Studio 2015 and a default pre-existing ASP.NET Core Web Application (.NET Framework) project called "MyBootstrapMVCApp".

1. Download

Before proceeding with the tutorial, download the jQWidgets Bootstrap Tag Helpers package and unpack it in a location of your choosing.

2. Add the jQWidgets Bootstrap Tag Helpers Project

In order to make the jQWidgets Bootstrap Tag Helpers available in our project we will have to add the "jQWidgets_Bootstrap_tag_helpers" project to the solution of "MyBootstrapMVCApp".

Right-click the src folder and select Add → Existing Project...:

Browse for the folder you extracted the jQWidgets Bootstrap Tag Helpers package to and select the file jQWidgets_Bootstrap_tag_helpers.xproj. It will be added to the solution.

3. Reference the jQWidgets Bootstrap Tag Helpers Project

Right-click the "MyBootstrapMVCApp" References and select Add Reference... In the Reference Manager pop-up window, check "jQWidgets_Bootstrap_tag_helpers":

4. Update _ViewImports.cshtml

Open the folder Views and then the file _ViewImports.cshtml. Add the following line to the bottom:

@addTagHelper *, jQWidgets_Bootstrap_tag_helpers

This way the jQWidgets Bootstrap Tag Helpers are now "registered" for use.

5. Add a reference to bootstrap.mvc.css

In the jQWidgets Bootstrap Tag Helpers download package you can also find the file bootstrap.mvc.css. It has to be referenced on the page where the Tag Helpers will appear. In our case, we add the file to the folder wwwroot\css:

and reference it in the file Views\Shared\_Layout.cshtml:

<environment names="Development">
    <link rel="stylesheet" href="~/lib/bootstrap/dist/css/bootstrap.css" />
    <link rel="stylesheet" href="~/css/site.css" />
    <link rel="stylesheet" href="~/css/bootstrap.mvc.css" />
</environment>

6. Build the Solution

Press F6 to Build the solution.

7. Add a Bootstrap Tag Helper to a Page

We can now add a Bootstrap Tag Helper to one of the "MyBootstrapMVCApp" views (pages). For example, we will add the <bootstrap-alert> Tag Helper to Views\Home\About.cshtml. While typing, IntelliSense suggest the existing Tag Helpers:

If we add the following lines of code:

<bootstrap-alert class="alert-warning alert-dismissible fade in">
    <bootstrap-alert-close><span aria-hidden="true">×</span></bootstrap-alert-close>
    <strong>Holy guacamole!</strong> Best check yo self, you're not looking too good.
</bootstrap-alert>
