Introduction to ASP.NET MVC 6 Tag Helpers

Tag Helpers are a feature introduced in ASP.NET MVC 6. The purpose of Tag Helpers is to enable server-side code to participate in creating and rendering HTML elements in Razor files.

Tag Helpers have some key advantages in contrast to HTML Helpers, the previous approach to server-side creation of markup in Razor views. Some of them are:

  • Razor markup using Tag Helpers looks like standard HTML;
  • IntelliSense support;
  • Useful built-in Tag Helpers;
  • Custom Tag Helpers can be implemented.
